# 🏠 Умный Дом — Готовые решения для жизни и бизнеса

https://ibrahimbakry.github.io/SmartHome/

Адаптивный лендинг для презентации готовых пакетных решений умного дома на базе Zigbee. Сайт представляет два направления: для владельцев жилья и для застройщиков / строительных компаний.

## 📋 О проекте

Этот сайт создан для демонстрации услуг по установке умного дома. Главная особенность — **готовые пакетные решения** с фиксированной ценой, что упрощает выбор для клиента.

### ✨ Возможности

- 🔄 **Две версии сайта**: для владельцев жилья и для застройщиков
- 🎨 **4 цветовые темы**: технологичная, тёплая, природная, ночная
- 📱 **Адаптивный дизайн** — отлично смотрится на всех устройствах
- 🧩 **Телескопическое меню** — компактное управление слева
- 💾 **Сохранение настроек** — тема и страница запоминаются в localStorage

## 🚀 Технологии

- **HTML5** — структура страниц
- **CSS3** — стилизация, анимации, адаптивность
- **JavaScript** — переключение страниц, смена темы, интерактивность
- **Font Awesome 6** — иконки
- **Google Fonts (Inter)** — современный шрифт

## 🎯 Целевая аудитория

### 👨‍👩‍👧‍👦 Для владельцев жилья
Простые и понятные пакеты:
- **Свет** — управление освещением + датчик движения в подарок
- **Безопасность** — охрана периметра + уведомления
- **Кухня** — управление техникой через розетки
- **Всё включено** — полная автоматизация + камера в подарок

### 🏢 Для застройщиков и строителей
Бизнес-пакеты с расчётом ROI:
- **Эконом** — для студий и 1-комнатных квартир
- **Оптимальный** — для 2-3 комнатных квартир
- **Премиум** — для элитного жилья
- **Ремонт** — для капремонта без штробления

## 🛠️ Установка и запуск

### Локальный запуск
1. Скачайте репозиторий:
   git clone https://github.com/ibrahimbakry/SmartHome.git
2. Откройте `index.html` в любом современном браузере

### Публикация на GitHub Pages
1. Загрузите файлы в репозиторий
2. В настройках репозитория включите GitHub Pages (ветка `main`)
3. Сайт будет доступен по адресу: `https://yourusername.github.io/your-repo-name/`
