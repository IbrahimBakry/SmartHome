## 🔧 Настройка под себя

### Изменение цен и состава пакетов
В файле `index.html` найдите соответствующие карточки и измените:
- Цены в блоках `price-tag`
- Списки оборудования в `equip-list`
- Описания сценариев в `scenario-box`

### Добавление/изменение темы
В файле `style.css` найдите блоки `body.theme-*` и измените CSS-переменные:

```css
body.theme-mytheme {
    --accent: #ваш_цвет;
    --accent-light: #ваш_светлый_цвет;
    /* остальные переменные */
}
```
